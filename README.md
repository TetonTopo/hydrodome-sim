# HydroDome V4 Simulation

Interactive wildfire defense simulation for Fort St. James, BC.

**Live Demo:** https://tetontopo.github.io/hydrodome-sim/

## About

HydroDome is a permanent wildfire defense infrastructure system by Carmanah Wildfire. This simulation demonstrates zone-based tower deployment with:

- 409 mapped roads
- Dynamic water boundary integration
- Fire spread modeling with ember shower mechanics
- Sprinkler coverage visualization
- Autonomous activation sequences

## Development

Built as a standalone HTML/JavaScript application. Edit `index.html` and push to see changes live on GitHub Pages.
